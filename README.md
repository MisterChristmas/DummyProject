# Installation
1 - Ouvrir la console en bash

3 - npm install
4 - ng add @angular/material
4.1 - Faire yes à tout, choisir le thème souhaité (de toute façon il est override par le thème custom)
5 - ng serve
6 - Ouvrir Localhost:4200
